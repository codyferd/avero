Avero 1.4.3

Avero is an open source, web-based operating environment that is based on Vue and TailwindCSS as
its core libraries. Avero is meant to be easy and user-friendly, allowing for people to use the
many provided "applets". Avero is my personal project, and you can find more about me in the
"About" applet. Avero updates every sunday in a weekly release. If you need more information,
contact me with my email, which is lucarappferdinand@gmail.com. Thank you for using Avero.
