Avero 1.15.1

Avero is an open source, web-based operating environment that is based on Vue and TailwindCSS as
its core libraries. Avero is meant to be easy and user-friendly, allowing for people to use the
many provided apps. Avero is my personal project, and you can find more about me in the
"About" app. Avero updates every sunday in a weekly release, along with minor updates in between.
If you need more information, contact me with my email, which is lucarappferdinand@gmail.com. Thank
you for using Avero.
